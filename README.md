【升级指导】
1、替换Automation.base目录下所有文件；
2、替换Automation.utils目录下所有问题；（如果__init__.py存在自定义类，请自行合入）
3、替换Automation.__init__.py文件；
4、Automation.resource目录：
  3.1、创建log文件夹；
  3.2、将app_data文件夹及其内容合入。
